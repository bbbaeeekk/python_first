vscode를 열었는데 터미널이 안보여요!

=>vscode 메뉴의 터미널 메뉴=> 새터미널

vscode의 터미널을 파워쉘이 아니라 GIt bash로 바꾸기.

1. '+'버튼 누르고 기본 프로필 설정
2. 화면 위쪽에 선택 가능한 프로그램 목록이 뜨는데 거기서 Git bash 선택(Git 이 설치 되어 있어야 한다.)
3. 기존의 파워쉘 터미널은 휴지통버튼을 통해 제거
4. 다시 새미널을 생성해서 bash로 되어있는지 확인.
